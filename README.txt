SK Polymers simple website
Files:
- index.html : single-file responsive website (HTML + CSS + JS)
How to use:
1. Unzip and open index.html in a browser.
2. To use with a real contact backend, replace the form handler JS with AJAX to your API or integrate services like EmailJS, Formspree, or your server endpoint.
3. To change phone/email/company details edit index.html.
